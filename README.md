# brain-tumor-detection-using-YOLO
## Overview
This project focuses on detecting brain tumors from MRI images using YOLO models.  
It was developed as part of a research study comparing different YOLO versions.

## Models Used
- YOLOv5
- YOLOv8
- YOLOv11

## Key Results
YOLOv8 achieved the best performance, with high detection accuracy (mAP@0.5 ≈ 0.924). 

## Tools
- Python
- YOLO
- Google Colab
